# TANGER CODE — Frontend

Site web vitrine Next.js 14+ pour TANGER CODE — développement web & mobile sur mesure.

## Stack

- **Next.js 14+** (App Router), **TypeScript** strict
- **Tailwind CSS** — palette "Mediterranean Tech"
- **Shadcn/ui** — composants UI
- **Next-Intl** — internationalisation (FR, EN, AR)
- **Next-Themes** — dark/light mode (dark par défaut)
- **Framer Motion** — animations
- **Fonts** : Cabinet Grotesk (titres), Inter (body), JetBrains Mono (code), Cairo (arabe)

## Quick start

```bash
# From project root:
cd ..
docker-compose up

# Or locally (without Docker):
npm install
npm run dev
```

Visit:
- `http://localhost:3000/fr` — French (default)
- `http://localhost:3000/en` — English
- `http://localhost:3000/ar` — Arabic (RTL)

## Commands

```bash
npm run dev        # Dev server
npm run build      # Production build
npm run start      # Start production server
npm run lint       # ESLint
npm run format     # Prettier
npm run typecheck  # TypeScript check
```
